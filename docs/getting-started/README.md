# Getting Started

Start here if you're new to CBass.

## Guides

| Guide | Time | Description |
|-------|------|-------------|
| [Quick Start](./quick-start.md) | 5 min | Get CBass running |
| [First Workflow](./first-workflow.md) | 10 min | Build your first AI automation |
| [Biology Projects](./biology-projects.md) | Varies | Apply AI to biology learning |

## Learning Path

### Day 1: Setup
1. Follow [Quick Start](./quick-start.md)
2. Create accounts in n8n and Open WebUI
3. Chat with the AI in Open WebUI

### Day 2: First Workflow
1. Complete [First Workflow](./first-workflow.md)
2. Modify the workflow for biology questions
3. Test with different prompts

### Week 1+: Projects
1. Pick a project from [Biology Projects](./biology-projects.md)
2. Start simple, add complexity
3. Combine multiple services
